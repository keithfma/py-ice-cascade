py_ice_cascade.main module¶

Python ICE-CASCADE combined glacial-fluvial-hillslope landscape evolution model

class py_ice_cascade.main.main_model(hillslope, uplift, x, y, z_rx, time_start, time_step, num_steps, out_steps)[source]¶

Composite landscape evolution model. Integrates glacial, fluvial, and hillslope model components and handles input-output.

Parameters:
  • hillslope – initialized hillslope model component, expect child of py_ice_cascade.hillslope.base_model class
  • uplift – initialized uplift model component, expect child of py_ice_cascade.uplift.base_model class
  • x – numpy vector, x-coordinate, [m]
  • y – numpy vector, y-coordinate, [m]
  • z_rx – grid, initial bedrock elevation, [m]
  • time_start – scalar, starting time, [a]
  • time_step – scalar, topographic model time step, [a]
  • num_steps – scalar, total steps in simulation, i.e. duration, [1]
  • out_steps – list, step numbers to write output, 0 is initial state, [1]
run(file_name, clobber=False)[source]¶

Run model simulation, save results to file

Parameters:
  • file_name – String, path to which results should be saved
  • clobber – Boolean, allow overwriting output file

Module contents¶

Python ICE-CASCADE landscape evolution model

Note: simulation progress is reported using the logging module, with the level set at the package level logger py_ice_cascade.logger. The logging level can be modified by explicitly updating this logger, like so:

import py_ice_cascade
import logging
py_ice_cascade.logger.setLevel(logging.WARNING)
