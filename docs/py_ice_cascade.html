py_ice_cascade.example module¶

Example cases for for Python ICE-CASCADE glacial-fluvial-hillslope landscape evolution model.

Examples can be run either directly in the python interpreter, e.g.:

import py_ice_cascade
py_ice_cascade.example.hill_only

or from the command line by executing the module as a script, e.g.:

python -m py_ice_cascade.example -c hill_only

In the latter case, use the -h flag to get additional help

class py_ice_cascade.hillslope.ftcs(height, delta, kappa, bc)[source]¶

Hillslope diffusion model using forward-time center-space (FTCS) finite diffence scheme.

Parameters:
  • height – 2D numpy array, surface elevation in model domain, [m]
  • delta – Scalar double, grid spacing, assumed square, [m]
  • kappa – 2D numpy array, diffusion coefficient, [m**2/a]
  • bc – List of boundary conditions names for [y=0, y=end, x=0, x=end]

Supported boundary conditions are:

  • constant: \(\frac{\partial H}{\partial t} = 0\)
  • closed: no flux out of boundary (e.g. \((q_x)_{i,j+1/2} = 0\) at \(x_{max}\))
  • open: no flux gradient normal to boundary, material passes through (e.g. \(\frac{\partial q_x}{\partial x} = 0\) at \(x_{max}\))
  • cyclic: flux at opposing boundaries is equal (e.g. \((q_x)_{i,-1/2} = (q_x)_{i,\text{end}+1/2}\))
  • mirror: boundary flux is equal and opposite incoming flux (e.g. \((q_x)_{i,j+1/2} = -(q_x)_{i,j-1/2}\) at \(x_{max}\))

Overview of FTCS scheme with spatially variable diffusivity (see reference (1) and (2)). Starting with the basic diffusion equation:

\[\frac{\partial H}{\partial t} = \nabla \cdot q\]

Discretize derivative terms using FTCS scheme:

\[\frac{H^{n+1}_{i,j} - H^{n}_{i,j}}{\Delta_t} = \frac{(q_x)^{n}_{i,j+1/2} - (q_x)^{n}_{i,j-1/2}}{\Delta_x} + \frac{(q_y)^{n}_{i+1/2,j} - (q_y)^{n}_{i-1/2,j}}{\Delta_y}\]

Expand the fluxes in terms of height gradient:

\[\begin{split}\frac{1}{\Delta_t} \left( H^{n+1}_{i,j} - H^{n}_{i,j} \right) = \frac{1}{\Delta_x} \left[ \kappa_{i,j+1/2} \frac{H^{n}_{i,j+1}-H^{n}_{i,j}}{\Delta_x} - \kappa_{i,j-1/2} \frac{H^{n}_{i,j}-H^{n}_{i,j-1}}{\Delta_x} \right] + \\ \frac{1}{\Delta_y} \left[ \kappa_{i+1/2,j} \frac{H^{n}_{i+1,j}-H^{n}_{i,j}}{\Delta_y} - \kappa_{i-1/2,j} \frac{H^{n}_{i,j}-H^{n}_{i-1,j}}{\Delta_y} \right]\end{split}\]

Use the mean diffusivity at midpoints:

\[\begin{split}\frac{1}{\Delta_t} \left( H^{n+1}_{i,j} - H^{n}_{i,j} \right) = \frac{1}{\Delta_x} \left[ \frac{\kappa_{i,j+1} + \kappa_{i,j}}{2} \frac{H^{n}_{i,j+1}-H^{n}_{i,j}}{\Delta_x} - \frac{\kappa_{i,j} + \kappa_{i,j-1}}{2} \frac{H^{n}_{i,j}-H^{n}_{i,j-1}}{\Delta_x} \right] + \\ \frac{1}{\Delta_y} \left[ \frac{\kappa_{i+1,j} + \kappa_{i,j}}{2} \frac{H^{n}_{i+1,j}-H^{n}_{i,j}}{\Delta_y} - \frac{\kappa_{i,j} + \kappa_{i-1,j}}{2} \frac{H^{n}_{i,j}-H^{n}_{i-1,j}}{\Delta_y} \right]\end{split}\]

Assume equal x- and y- grid spacing and collect coefficients:

\[\begin{split}\frac{1}{\Delta_t} \left( H^{n+1}_{i,j} - H^{n}_{i,j} \right) = \frac{1}{2 \Delta_{xy}^2} \left[ (\kappa_{i,j+1} + \kappa_{i,j}) (H^{n}_{i,j+1}-H^{n}_{i,j}) - (\kappa_{i,j} + \kappa_{i,j-1}) (H^{n}_{i,j}-H^{n}_{i,j-1}) + \\ (\kappa_{i+1,j} + \kappa_{i,j}) (H^{n}_{i+1,j}-H^{n}_{i,j}) - (\kappa_{i,j} + \kappa_{i-1,j}) (H^{n}_{i,j}-H^{n}_{i-1,j}) \right]\end{split}\]

Expand and collect terms:

\[\begin{split}H^{n+1}_{i,j} = H^{n}_{i,j} + \frac{\Delta_t}{2 \Delta_{xy}^2} \left[ H^{n}_{i,j+1} (\kappa_{i,j+1} + \kappa_{i,j}) + H^{n}_{i,j-1} (\kappa_{i,j} + \kappa_{i,j-1}) + H^{n}_{i+1,j} (\kappa_{i+1,j} + \kappa_{i,j}) + \\ H^{n}_{i-1,j} (\kappa_{i,j} + \kappa_{i-1,j}) - H^{n}_{i,j} (4 \kappa_{i,j} + \kappa_{i,j+1} + \kappa_{i,j-1} + \kappa_{i+1,j} + \kappa_{i-1,j}) \right]\end{split}\]

The above scheme is modified at boundary points.

Composite landscape evolution model. Integrates glacial, fluvial, and hillslope model components and handles input-output.

Parameters:

  • y – numpy vector, y-coordinate, [m]
  • zrx – grid, initial bedrock elevation, [m]
  • time_start – scalar, starting time, [a]
  • time_step – scalar, topographic model time step, [a]
  • num_steps – scalar, total steps in simulation, i.e. duration, [1]
  • out_steps – list, step numbers to write output, 0 is initial state, [1]
  • hill_on – scalar bool, set True to enable hillslope model
  • hill_kappa_active – scalar, hillslope diffusivity where active, [m^2 / a]
  • hill_kappa_inactive – scalar, hillslope diffusivity where inactive, [m^2 / a]
  • hill_bc – list, hillslope model boundary conditions at [y[0], y[end], x[0], x[end]. See hilllslope.py for details.
  • uplift_on – scalar bool, set True to enable uplift component model
  • uplift_start – grid, uplift rate at time_start [m / a]
  • uplift_end – grid uplift rate at time_end [m / a]
  • verbose – Boolean, set True to show verbose messages

class py_ice_cascade.uplift.linear(ui, uf, ti, tf)[source]¶

Tectonic uplift model in which uplift is linearly interpolated between a pre-defined initial and final state.

Parameters:
  • ui – 2D numpy array, initial uplift rate, [m/a]
  • uf – 2D numpy array, final uplift rate, [m/a]
  • ti – Scalar, initial time, [a]
  • tf – Scalar, final time, [a]

Let \(u(x,y,t)\) be the uplift rate as a function of space (\(x,y\)) and time (\(t\)), and let the initial and final time be \(t_i, t_f\), respectively. This class defines \(u\) as:

\[u(x,y,t) = u(x,y,t_i) + \frac{u(x,y,t_f) - u(x,y,t_i)}{t_f-t_i} \left( t-t_i \right)\]

Omiting the \(x,y\) coordinates:

\[u(t) = u(t_i) + \frac{u(t_f) - u(t_i)}{t_f-t_i} (t - t_i) = u_i + \frac{u_f - u_i}{t_f - t_i} (t - t_i)\]

It is useful redefine the slope term as \(b\) and gather up constants into a new term \(a\), like so:

\[u(t) = u_i + b(t - t_i) = (u_i - b t_i) + b t = a + b t\]

The method get_uplift_rate() returns the above uplift rate. The method get_uplift() instead returns the total uplift over some time interval \([t_{start},t_{end}]\). This is simply the definite integral of the uplift rate:

\[\begin{split}\int_{t_{start}}^{t_{end}} (a + b t) dt &= (a t + \frac{1}{2} b t^2 + c)|_{t_{end}} - (a t + \frac{1}{2} b t^2 + c)|_{t_{start}} \\ &= (a t_{end} + \frac{1}{2} b t_{end}^2) - (a t_{start} + \frac{1}{2} b t_{start}^2) \\ &= a(t_{end}-t_{start}) + \frac{b}{2} (t_{end}^2 - t_{start}^2)\end{split}\]

Since \(a\) and \(b\) are constant coefficients, they are precomputed for efficiency.
